# SQL Data Validation & Visualisation – Northwind Retail Database

## Project Overview
This project demonstrates **data validation and business insight generation** using the Northwind sample retail database.  
It combines:
- **SQL integrity checks** (duplicates, foreign keys, missing data)  
- **Summary reports** (sales by region, top customers)  
- **Visualization in Tableau** for clear business communication  

Outcome: A reproducible workflow that bridges **technical rigor** with **business-ready dashboards**.

---

## Tools & Technologies
- **Google Colab** (Python, Pandas, SQLite)
- **SQL** (validation queries, joins, aggregations)
- **Tableau Public** (interactive dashboards)
- **GitHub** (documentation & reproducibility)

---

## Data Validation Steps
- Checked for duplicate customer records  
- Validated foreign keys between Customers and Orders  
- Generated summary reports (sales by region, top customers)
